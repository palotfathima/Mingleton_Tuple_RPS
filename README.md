A simple command-line Rock, Paper, Scissors game written in Python!

#How the Game Works?
Pretty straightforward. The player selects one of three options: 1 → Rock, 2 → Paper and 3 → Scissors.
The computer randomly selects its weapon, and the winner is decided based on the standard Rock-Paper-Scissors rules.

#How I did I do this?
Python loops and conditional statements, and some good-ol random number generation.
